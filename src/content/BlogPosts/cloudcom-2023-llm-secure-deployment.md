# Presented at IEEE CloudCom 2023

I presented the following paper at **IEEE International Conference on Cloud Computing Technology and Science (CloudCom 2023)**.

**Flexible and Secure Code Deployment in Federated Learning using Large Language Models: Prompt Engineering to Enhance Malicious Code Detection**

Authors: **Jungwon Seo**, **Nan Zhang**, **Chunming Rong**

Conference details:
- Event: **IEEE CloudCom 2023**
- Dates: **December 4-6, 2023**
- Location: **Naples, Italy**

- [Paper (IEEE Xplore)](https://ieeexplore.ieee.org/abstract/document/10475813)

![CloudCom 2023 FL secure deployment workflow](/images/news/cloudcom-llm-secure-deployment.png)

## Brief Summary

This work explores how prompt engineering with large language models can strengthen malicious code detection during federated learning code deployment, aiming to improve both flexibility and security in practical FL workflows.
